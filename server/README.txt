To get started, enter this directory and create a virtual environment if it does not exit
$ virtualenv venv --python=python3

Start the environment with
$ source venv/bin/activate

Install Django if the environment is new
$ pip install Django

Run the server with
$ python server/manage.py runserver

Exit the environment with
$ deactivate


Instructions above taken from https://uofa-cmput404.github.io/lab-4-django.html
